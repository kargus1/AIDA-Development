TopBarUiState

data class TopBarUiState(var isGestureEnabled: Boolean = false, var isMicEnabled: Boolean = false, var isCameraEnabled: Boolean = false, val isLidarEnabled: Boolean = false, val connectionStates: Map<ConnectionId, ConnectionState> = ConnectionId.entries.associateWith { ConnectionState.Disconnected })

Represents the UI state for the application's top bar. This includes the enabled status of various features like gestures, microphone, camera, Lidar, and the connection states of different robot components.
