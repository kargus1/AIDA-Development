ManualControlViewModel

class ManualControlViewModel @Inject constructor(manualRepository: ManualControlRepository, settingsRepository: AppPrefsRepository, connectionRepository: ConnectionRepository, topBarRepository: TopBarRepository) : ViewModel

ViewModel for the Manual Control screen. Manages the state and business logic for manual robot interaction, including video streaming, Lidar data, joystick control, and speech-to-text functionality. It combines data from various repositories to update the ManualControlUiState.

Parameters

manualRepository

Repository for handling manual control commands and data flows (STT, video, Lidar, joystick).

settingsRepository

Repository for accessing application preferences like IP address and port.

connectionRepository

Repository for managing and observing connection states to the robot.

topBarRepository

Repository for observing top bar states, such as microphone and camera enabled status.
