TopBarViewModel

class TopBarViewModel @Inject constructor(topBarRepository: TopBarRepository, connectionRepository: ConnectionRepository) : ViewModel

ViewModel for the application's top bar. Manages the UI state related to top bar controls (microphone, camera, gestures) and reflects overall connection statuses. It interacts with TopBarRepository to send commands and ConnectionRepository to observe connection states.

Repository for managing top bar feature states (mic, camera, gestures).
