SequenceBar

@Composable
fun SequenceBar(viewModel: SequenceViewModel, coroutineScope: CoroutineScope, uiState: SequenceBarState, scrollState: ScrollState, snapToClosestAction: () -> Unit)

Displays a horizontal "sequence bar" where users can arrange and configure actions.

This composable shows a row of actions that can be reordered, edited via double-tap or a settings button, and removed. It also includes a scrollable timeline and a progress bar to indicate the current position when scrolling horizontally.

Parameters

viewModel

The SequenceViewModel that contains business logic functions.

scrollState

The ScrollState controlling horizontal scrolling of the bar.

coroutineScope

The CoroutineScope used to launch animations or asynchronous tasks.

uiState

The SequenceBarState that contains the state for the SequenceBar such as the contained actions.

snapToClosestAction

A lambda function that snaps to the action closest to tbe center.
